Navigation Tests

[FUNDAMENTAL] MOA-NAV-010 Previous/Next Phrase

Tests whether previous/next phrase navigation is supported.

While Media Overlays (MO) are playing, use the next/previous phrase navigation control/command in the reading system with assistive technology to move through the next three paragraphs. The paragraph 1 below is one audio phrase. The sentence 1 in the next paragraph is also a single audio phrase. In sentence 2 each word is a separate audio phrase. Sentence 3 and paragraph 3 are also separate audio phrases.

This is paragraph 1.

This is sentence 1 within paragraph 2. This sentence 2 is divided into individual words. This is sentence number 3.

This is paragraph 3.

If previous/next phrase navigation plays media overlay at the level of sentences and words as described above, then the reading system supports media overlay playback at the granularity of audio phrases and this test passes.

[FUNDAMENTAL] MOA-NAV-020 Navigate to previous section

Tests whether navigation to previous section is supported.

Use any available control/command to move to the previous section. If playback starts at the section "[FUNDAMENTAL] MOA-NAV-010 Previous/Next Phrase" automatically or on using the Play control/command, the test passes.

[FUNDAMENTAL] MOA-NAV-030 Navigate to next section

[FUNDAMENTAL] MOA-NAV-070 Text and media overlay synchronization

Tests whether media overlay is synchronized with text display.

While Media Overlays (MO) are playing, perform the following steps:

  1. Move to the next section titled "[FUNDAMENTAL] MOA-NAV-080 Set and navigate to bookmarks"
  2. Stop media overlay playback
  3. Check the highlighted text on the screen. If a Magnifier is being used for evaluation, check the text in the Magnifier focus. If a Screen Reader is being used for evaluation, listen to the screen reader voice to determine the highlighted text. On Reading Systems without screen, the in-built TTS can be used to read the text of the book.

If the media overlay playback matches the highlighted text, the test passes.

[FUNDAMENTAL] MOA-NAV-080 Set and navigate to bookmarks

Tests whether bookmarks can be set and navigated to while media overlay is playing.

While Media Overlays (MO) are playing, set a bookmark at this location (Screen reader users may pause the playback, if they find it difficult to listen to the playback along with the screen reader voice). Then use the TOC to navigate away from this location e.g. choose the section "Basic Tests" in the TOC to move to it. Then use the bookmark list or "go to bookmark" feature in the reading system to move to the bookmark just set by you. If media overlay playback starts at the correct location, this test passes.

[FUNDAMENTAL] MOA-NAV-090 Section Navigation - TOC

Tests whether navigation to a new section is supported.

Go to the table of contents and select "Section Navigation, Part 2" from the list. If playback begins or can be started on the destination page, starting from "Section Navigation, Part 2", the test passes.
