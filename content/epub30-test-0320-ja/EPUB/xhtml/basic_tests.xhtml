Basic Tests

[FUNDAMENTAL] MOA-BASIC-010 Media Overlays Playback

Media Overlaysがサポートされているかどうかのテスト

If users can get information that playback is available after this page has been rendered, and if playback can be started and paused with assistive technology, then this test passes.

[FUNDAMENTAL] MOA-BASIC-020 Pause and resume from the same position, without closing the book

Tests whether media overlay playback resumes at correct location after pause

Pause the media overlay playback and note the last read position. Now start media overlay playback again and if playback starts at the last read location the test passes.

[FUNDAMENTAL] MOA-BASIC-030 Resume playback from last position on opening a book

Tests whether playback resumes even when the book is closed and opened again.

Pause media overlay playback, note the last read position and close the reading system. Open this book again in reading system and note the playback start position. If playback resumes at the last read position, the test passes.
