EPUB 3 Conformance Test Document: Media Overlay Fundamental Accessibility Tests 0301

Abstract

This document focuses on testing accessibility of an EPUB3 file with audio media overlays.

Status of this Document

This publication is currently considered [stable] by the IDPF and DAISY Consortium.

Before using this publication to evaluate reading systems, testers are strongly encouraged to verify that they have the latest release by checking the current release version and date of the test suite at www.epubtest.org

This publication is one of several that will comprise the EPUB 3 Accessibility Test Suite for reflowable content.

About this Document

This document contains tests which are fundamental to accessibility of the Reading System for users with disabilities when using an EPUB 3 with media overlays. The advanced tests are contained in a separate document.

The Reading Systems should be evaluated using different combinations of assistive technology and all the tests given in this book should be performed. It is expected that an accessible Reading System will pass all the tests given in this book regardless of the input and output mediums used in testing. Testers may perform the tests given in this book using keyboard, mouse or touch/gestures as input and screen reader/self voicing, visual adjustments or Braille as output.

Conventions

The following conventions are used throughout the document:

All tests in this document are preceded by the label [FUNDAMENTAL] indicating that the test is fundamental to establish accessibility of the Reading System.

The test labels are followed by ID such as MOA-BASIC-010. These test IDs correspond to the entries in the form on www.epubtest.org for accessibility evaluation.

Performing the tests

The result of each test in this book should be recorded in the corresponding field in the form on the website and comments if any should be noted in the space provided in the online form.
